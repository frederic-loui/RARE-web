# Example: static routing with interface

## **Topology diagram**

![topology](/img/rout-static07.tst.png)

## **Configuration**

**r1:**
```
hostname r1
buggy
!
logging file debug ../binTmp/zzz-log-r1.run
!
vrf definition v1
 rd 1:1
 exit
!
interface ethernet1
 no description
 vrf forwarding v1
 ipv4 address 1.1.1.1 255.255.255.0
 ipv6 address 1234::1 ffff:ffff::
 no shutdown
 no log-link-change
 exit
!
interface ethernet2
 no description
 vrf forwarding v1
 ipv4 address 1.1.1.1 255.255.255.0
 ipv6 address 1234::1 ffff:ffff::
 no shutdown
 no log-link-change
 exit
!
!
ipv4 route v1 1.1.1.2 255.255.255.255 1.1.1.2 interface ethernet1
ipv4 route v1 1.1.1.3 255.255.255.255 1.1.1.3 interface ethernet2
!
ipv6 route v1 1234::2 ffff:ffff:ffff:ffff:ffff:ffff:ffff:ffff 1234::2 interface ethernet1
ipv6 route v1 1234::3 ffff:ffff:ffff:ffff:ffff:ffff:ffff:ffff 1234::3 interface ethernet2
!
!
!
!
!
!
!
!
!
!
!
!
end
```
